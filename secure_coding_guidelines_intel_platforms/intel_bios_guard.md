### Intel速 Bios Guard {#intel-bios-guard}

**#BiosGuard.1**: Intel速 Bios Guard SHOULD be used for a firmware update.

**#BiosGuard.2**: The platform MUST make sure all the processes are in SMM when performing the Intel速 Bios Guard update.

**#BiosGuard.3**: Once Intel速 Bios Guard is enabled, there MUST be no way to disable it.
